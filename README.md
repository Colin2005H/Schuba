# Schuba

## Presentation

Schuba est une application web utilisant Laravel, qui permet la gestion des formations, séances et niveau dans un club de plongée.

## Installation

### Étape 1

```
composer require "darkaonline/l5-swagger"
```

```
composer install
```

### Étape 2

*copier le*
```
.env.example
```

*renommer en*
```
.env
```

### Étape bonus

*pour lancer*
```
php artisan serve
```
